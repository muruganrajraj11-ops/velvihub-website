📖 What is EtherChannel?

EtherChannel என்பது பல Physical Links-ஐ ஒரு Logical Link ஆக இணைக்கும் Technology ஆகும்.

EtherChannel = Multiple Links → Single Logical Link

🚀 Benefits

✔ Increased Bandwidth

✔ Load Balancing

✔ Redundancy

✔ High Availability

🔧 EtherChannel Protocols

🔹 PAgP (Cisco Proprietary)

🔹 LACP (IEEE 802.3ad)

interface range fa0/1 - 2

channel-group 1 mode active
