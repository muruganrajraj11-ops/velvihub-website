🌐 What is Subnet Mask? – தமிழில் எளிமையாக

Subnet Mask என்பது ஒரு IP Address-ல் எந்த பகுதி Network மற்றும் எந்த பகுதி Host என்பதை பிரிக்கும் Addressing Tool ஆகும்.

🏠 Real-Life Example

ஒரு Apartment Address-ல்: Block A = Network Flat No 101 = Host என்பது போல Subnet Mask IP Address-ஐ பிரிக்கிறது.

📶 Example

IP Address = 192.168.1.10

Subnet Mask = 255.255.255.0

இதில் 192.168.1 என்பது Network பகுதி. 10 என்பது Host பகுதி.

✅ ஏன் Subnet Mask தேவை?

ஒரே Network-ல் இருக்கும் Devices யாரெல்லாம் என்பதை கண்டுபிடிக்க உதவுகிறது.

🎯 முடிவு

Subnet Mask என்பது Network மற்றும் Host பகுதியை பிரிக்கும் மிகவும் முக்கியமான Networking Concept ஆகும்.
