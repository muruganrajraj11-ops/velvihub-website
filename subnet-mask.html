🌐 What is Subnet Mask? – தமிழில் எளிமையாக

Subnet Mask என்பது ஒரு IP Address-ல் எந்த பகுதி Network என்றும் எந்த பகுதி Host என்றும் பிரித்து காட்டும் Addressing Tool ஆகும்.

🏠 Real-Life Example

ஒரு Apartment Address-ல்:

Block A = Network

Flat No 101 = Host

என்பது போல Subnet Mask IP Address-ஐ பிரிக்கிறது.

📶 Example

IP Address = 192.168.1.10

Subnet Mask = 255.255.255.0

இதில் 192.168.1 என்பது Network பகுதி.

10 என்பது Host பகுதி.

📊 Common Subnet Masks

Subnet Mask CIDR Hosts
255.255.255.0 /24 254
255.255.0.0 /16 65,534
255.0.0.0 /8 16 Million+

✅ ஏன் Subnet Mask தேவை?

  • Network மற்றும் Host பகுதியை பிரிக்க
  • IP Address Planning செய்ய
  • Broadcast Traffic குறைக்க
  • Network Management எளிதாக்க

🎯 Interview Questions

Q: Subnet Mask என்றால் என்ன?

A: Network மற்றும் Host பகுதியை பிரிக்கும் Address.

Q: 255.255.255.0 எந்த CIDR?

A: /24

Q: Subnet Mask ஏன் பயன்படுத்தப்படுகிறது?

A: Network Identification மற்றும் IP Planning-க்காக.

🚦 Easy Analogy

ஒரு பெரிய Apartment-ல் Block Number மற்றும் Flat Number இருப்பது போல, Subnet Mask Network மற்றும் Host பகுதியை பிரிக்கிறது.

🎯 முடிவு

Subnet Mask என்பது Network மற்றும் Host பகுதியை பிரிக்கும் மிகவும் முக்கியமான Networking Concept ஆகும்.

IP Addressing, Routing மற்றும் Subnetting ஆகிய அனைத்திற்கும் இது அடிப்படையாக செயல்படுகிறது.
