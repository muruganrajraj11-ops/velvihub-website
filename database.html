📖 What is a Database?

Database என்பது தகவல்களை (Data) சேமித்து, நிர்வகித்து, தேவைப்படும் போது மீட்டெடுக்க உதவும் அமைப்பு ஆகும்.

Database = Organized Data Storage System

✔ Fast Retrieval

✔ Data Security

✔ Data Consistency

✔ Multi User Access

🔑 Primary Key

Primary Key என்பது ஒரு Table-ல் ஒவ்வொரு Record-ஐ தனித்தனியாக அடையாளம் காண பயன்படும் Unique Value ஆகும்.

📝 Create Table Example

CREATE TABLE students (

id INT,

name VARCHAR(50),

course VARCHAR(50)

);

➕ Insert Data

INSERT INTO students
VALUES (1,'Raj','Networking');

🔍 Select Data

SELECT * FROM students;

✏ Update Data

UPDATE students

SET course='Linux'

WHERE id=1;

❌ Delete Data

DELETE FROM students

WHERE id=1;
