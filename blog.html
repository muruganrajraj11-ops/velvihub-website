🌐 What is DNS? – தமிழில் எளிமையாக

DNS என்றால் Domain Name System. இது internet-ல website பெயரை IP address-ஆ மாற்றும் system.

உதாரணம்: நாம் browser-ல google.com type பண்ணுறோம். ஆனா computerக்கு பெயர் புரியாது. அதுக்கு number தான் புரியும். DNS அந்த பெயரை IP address-ஆ மாற்றி சரியான website-க்கு connect பண்ணும்.

Real-life example: உங்க mobileல ஒருத்தர பெயர் “Ravi”ன்னு save பண்ணி இருக்கீங்க. நீங்க Raviன்னு press பண்ணினா call போகும். உள்ளே phone number இருக்கும். அதே மாதிரி: Website name = google.com Phone number = IP address DNS = contact book

Google DNS example: 8.8.8.8 8.8.4.4

DNS இல்லனா website open ஆகாது. அதனால internet-ல ரொம்ப முக்கியமான service இதுதான்.

🚪 What is Gateway? – தமிழில் எளிமையாக

Gateway என்பது நம்ம device வெளியே internet-க்கு போகும் வழி.

உதாரணம்: வீட்டிலிருந்து வெளியே போக main gate use பண்ணுறோம். அதே மாதிரி mobile அல்லது laptop internet போக router gateway use பண்ணும்.

Router IP பொதுவா: 192.168.1.1 அல்லது 192.168.0.1

உங்க device: IP = 192.168.1.10 Gateway = 192.168.1.1

Internet போக எல்லா trafficயும் முதல்ல gatewayக்கு போகும். அதுக்கப்புறம் வெளியே செல்லும்.
