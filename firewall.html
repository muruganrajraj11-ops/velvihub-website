🛡️ Firewall என்றால் என்ன? – தமிழில் எளிமையாக

Firewall என்பது உங்கள் Network-ஐ பாதுகாக்கும் Security Device அல்லது Software ஆகும்.

வெளியிலிருந்து வரும் ஆபத்தான Traffic-ஐ தடுத்து, அனுமதிக்கப்பட்ட Traffic-ஐ மட்டும் உள்ளே அனுமதிக்கிறது.

🏠 Real-Life Example

ஒரு Apartment Gate-ல் Security Guard இருப்பார்.

அவர் யார் உள்ளே வர வேண்டும், யார் வரக்கூடாது என்பதை முடிவு செய்வார்.

அதேபோல் Firewall உங்கள் Network-க்கு வரும் Traffic-ஐ சோதித்து அனுமதிக்கிறது அல்லது தடுக்கிறது.

🚪 Firewall எப்படி வேலை செய்கிறது?

Network-க்குள் வரும் மற்றும் வெளியே செல்லும் Data Packets-ஐ Firewall ஆய்வு செய்கிறது.

Rules அடிப்படையில் அனுமதிக்க வேண்டுமா அல்லது Block செய்ய வேண்டுமா என்று முடிவு செய்கிறது.

📦 Packet Filtering

Source IP, Destination IP, Port Number போன்ற தகவல்களை பார்த்து Traffic-ஐ Allow அல்லது Deny செய்கிறது.

இது Firewall-ன் அடிப்படை பாதுகாப்பு முறை.

🔍 Stateful Inspection

Firewall தற்போதைய Connection-ஐ நினைவில் வைத்துக்கொண்டு Traffic சட்டபூர்வமானதா என்பதை சரிபார்க்கிறது.

இதனால் பாதுகாப்பு மேலும் அதிகரிக்கிறது.

🚀 Next Generation Firewall (NGFW)

Modern Firewall-கள் Malware Detection, Application Control, Intrusion Prevention போன்ற கூடுதல் பாதுகாப்பு வசதிகளை வழங்குகின்றன.

இவை Next Generation Firewall (NGFW) என்று அழைக்கப்படுகின்றன.

⚔️ Firewall vs Antivirus

Firewall Network Traffic-ஐ கட்டுப்படுத்துகிறது.

Antivirus கணினிக்குள் உள்ள Virus மற்றும் Malware-களை கண்டறிந்து நீக்குகிறது.

இரண்டும் வேறு வேறு பணிகளை செய்கின்றன.

🎯 Interview Questions

Q: Firewall-ன் முக்கிய பணி என்ன?

A: Unauthorized Access-ஐ தடுத்து Network-ஐ பாதுகாப்பது.


Q: NGFW என்பதன் விரிவாக்கம் என்ன?

A: Next Generation Firewall.

🚦 Firewall = Security Guard

ஒரு Security Guard Gate-ல் நின்று அனுமதி உள்ளவர்களை மட்டும் உள்ளே அனுப்புவார்.

அதேபோல் Firewall Network-க்கு அனுமதிக்கப்பட்ட Traffic-ஐ மட்டும் அனுமதிக்கிறது.

🎯 முடிவு

Firewall என்பது எந்த Network-க்கும் மிகவும் முக்கியமான பாதுகாப்பு அமைப்பாகும்.

Hackers, Malware மற்றும் Unauthorized Access-லிருந்து Network-ஐ பாதுகாக்க உதவுகிறது.
