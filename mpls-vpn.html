Layer 3 VPN, VRF & Service Provider Networks

📖 What is MPLS VPN?

MPLS VPN என்பது Service Provider Network மூலம் பல Customer Networks-ஐ பாதுகாப்பாக இணைக்கும் தொழில்நுட்பம்.

MPLS VPN = Private Network Over Shared Provider Network

🏢 VPN Components

CE Router (Customer Edge)

PE Router (Provider Edge)

P Router (Provider Router)

🔐 VRF

VRF (Virtual Routing and Forwarding) ஒவ்வொரு Customer-க்கும் தனி Routing Table வழங்குகிறது.

✔ Secure Communication

✔ Traffic Separation

✔ Scalability

✔ Multi-site Connectivity
