📖 IPv6 என்றால் என்ன?

IPv6 (Internet Protocol Version 6) என்பது IPv4-க்கு அடுத்த தலைமுறை IP Protocol ஆகும். IPv4 Address பற்றாக்குறையை தீர்க்க உருவாக்கப்பட்டது.

🤔 IPv6 ஏன் தேவை?

  • IPv4 Address தீர்ந்து வருகிறது
  • IoT Devices அதிகரிப்பு
  • சிறந்த Routing
  • Built-in Security
  • Future Ready Internet
