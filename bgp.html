🌍 BGP என்றால் என்ன? – தமிழில் எளிமையாக

BGP (Border Gateway Protocol) என்பது Internet-ல் பயன்படுத்தப்படும் மிக முக்கியமான Routing Protocol ஆகும்.

உலகம் முழுவதும் இருக்கும் ISP மற்றும் பெரிய Network-களை இணைக்க BGP பயன்படுகிறது.

🏠 Real-Life Example

சென்னையிலிருந்து அமெரிக்காவிற்கு ஒரு Parcel அனுப்ப வேண்டும் என்று வைத்துக்கொள்ளுங்கள்.

எந்த நாடுகள் வழியாக அனுப்பினால் சிறந்த பாதை கிடைக்கும் என்பதை Courier நிறுவனம் தேர்வு செய்யும்.

அதேபோல் Internet-ல் BGP சிறந்த Network Path-ஐ தேர்வு செய்கிறது.

⚙️ BGP எப்படி வேலை செய்கிறது?

ஒவ்வொரு பெரிய Network-க்கும் ஒரு Autonomous System (AS) Number இருக்கும்.

BGP Routers இந்த AS தகவல்களை பகிர்ந்து சிறந்த Route-ஐ தேர்வு செய்கின்றன.

📌 Autonomous System (AS)

ஒரே நிர்வாக கட்டுப்பாட்டில் இருக்கும் Network-களின் தொகுப்பு Autonomous System எனப்படும்.

உதாரணம்: ISP, Telecom Company, Cloud Provider.

🔢 ASN என்றால் என்ன?

ASN = Autonomous System Number

ஒவ்வொரு AS-க்கும் தனித்துவமான Number வழங்கப்படும்.

உதாரணம்: AS12345

🌐 eBGP vs iBGP

Type Meaning
eBGP Different AS இடையே
iBGP Same AS உள்ளே

📊 BGP vs OSPF

Feature BGP OSPF
Type Path Vector Link State

🎯 Interview Questions

Q: BGP என்பதன் விரிவாக்கம் என்ன?

A: Border Gateway Protocol

Q: BGP எந்த Port பயன்படுத்துகிறது?

A: TCP Port 179

Q: BGP எந்த வகை Protocol?

A: Path Vector Routing Protocol

🚦 Easy Analogy

Google Maps ஒரு நாட்டிலிருந்து இன்னொரு நாட்டிற்கு செல்லும் சிறந்த பாதையை தேர்வு செய்வது போல BGP Internet-ல் Network-களுக்கிடையே சிறந்த பாதையை தேர்வு செய்கிறது.

🎯 முடிவு

BGP என்பது Internet-ன் backbone Routing Protocol ஆகும்.

ISP, Cloud Provider மற்றும் Telecom Network-களில் இது மிகவும் முக்கியமானது.
