📖 What is Subnetting?

Subnetting என்பது ஒரு பெரிய Network-ஐ பல சிறிய Networks (Subnets) ஆக பிரிக்கும் செயல்முறை ஆகும்.

Subnetting = Divide One Network Into Multiple Smaller Networks

🎯 Why Subnetting?

✔ Better IP Utilization

✔ Reduced Broadcast Traffic

✔ Improved Security

✔ Easy Network Management

✔ Efficient Address Allocation

🌐 Subnet Mask

255.0.0.0 = /8

255.255.0.0 = /16

255.255.255.0 = /24

255.255.255.192 = /26

⚙ How Subnetting Works?

Host Bits-ஐ Borrow செய்து புதிய Subnets உருவாக்கப்படுகிறது.

ஒவ்வொரு Subnet-க்கும் தனி Network Address மற்றும் Broadcast Address இருக்கும்.

📊 Example

Network: 192.168.1.0/24

Subnet: /26

Subnet 1 : 192.168.1.0

Subnet 2 : 192.168.1.64

Subnet 3 : 192.168.1.128

Subnet 4 : 192.168.1.192
