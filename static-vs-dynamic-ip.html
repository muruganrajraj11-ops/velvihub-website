🔄 Static IP vs Dynamic IP – தமிழில் எளிமையாக

IP Address பொதுவாக இரண்டு வகைப்படும்:

  • Static IP Address
  • Dynamic IP Address

📌 Static IP என்றால் என்ன?

Static IP என்பது மாறாத IP Address ஆகும்.

ஒருமுறை Configure செய்தால் அதே IP எப்போதும் இருக்கும்.

உதாரணம்: 192.168.1.10

Server, CCTV, Printer மற்றும் Network Devices-ல் Static IP அதிகம் பயன்படுத்தப்படுகிறது.

🔄 Dynamic IP என்றால் என்ன?

Dynamic IP என்பது DHCP மூலம் தானாக வழங்கப்படும் IP Address ஆகும்.

இன்று: 192.168.1.101

நாளை: 192.168.1.105

என்று மாறக்கூடும்.

🏠 Real-Life Example

Static IP = உங்களுடைய வீட்டு நிரந்தர முகவரி.

Dynamic IP = Hotel Room Number.

இன்று ஒரு Room, நாளை வேறு Room கிடைக்கலாம்.

📊 Static IP vs Dynamic IP

Feature Static IP Dynamic IP
IP Address மாறாது மாறக்கூடும்
Configuration Manual Automatic (DHCP)
Best For Servers, CCTV Home Users
Management More Control Easy Management

✅ Static IP நன்மைகள்

  • Server Hosting-க்கு சிறந்தது
  • Remote Access எளிது
  • Network Devices-க்கு ஏற்றது
  • IP எப்போதும் மாறாது

✅ Dynamic IP நன்மைகள்

  • Automatic Configuration
  • Easy Management
  • Home Networks-க்கு ஏற்றது
  • DHCP மூலம் தானாக வழங்கப்படும்

🎯 Interview Questions

Q: Static IP என்றால் என்ன?

A: மாறாத IP Address.

Q: Dynamic IP யார் வழங்குவார்கள்?

A: DHCP Server.

Q: Server-க்கு எந்த IP சிறந்தது?

A: Static IP.

🚦 Easy Analogy

Static IP என்பது உங்கள் நிரந்தர வீட்டு முகவரி போன்றது.

Dynamic IP என்பது Hotel Room Number போன்றது.

தேவைக்கு ஏற்ப மாறிக்கொண்டே இருக்கும்.

🎯 முடிவு

Server, CCTV மற்றும் Network Infrastructure-க்கு Static IP சிறந்தது.

Mobile, Laptop மற்றும் Home WiFi பயன்பாட்டிற்கு Dynamic IP போதுமானது.
