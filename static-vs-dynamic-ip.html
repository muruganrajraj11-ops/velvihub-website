🔄 Static IP vs Dynamic IP – தமிழில் எளிமையாக

IP Address இரண்டு வகைப்படும்:

  • Static IP
  • Dynamic IP

📌 Static IP என்றால் என்ன?

Static IP என்பது மாறாத IP Address. ஒருமுறை configure செய்தால் அதே IP எப்போதும் இருக்கும்.

உதாரணம்: 192.168.1.10

Server, CCTV, Printer போன்றவற்றில் Static IP அதிகம் பயன்படுத்தப்படும்.

🔄 Dynamic IP என்றால் என்ன?

Dynamic IP என்பது DHCP மூலம் தானாக வழங்கப்படும் IP Address.

இன்று: 192.168.1.101

நாளை: 192.168.1.105

என்று மாறக்கூடும்.

🏠 Real-Life Example

Static IP = உங்களுடைய வீட்டு நிரந்தர முகவரி.

Dynamic IP = Hotel Room Number. இன்று ஒரு Room, நாளை வேறு Room.

⚖️ ஒப்பீடு

Static IP: ✔ மாறாது ✔ Server-க்கு சிறந்தது ✔ Manual Configuration

Dynamic IP: ✔ தானாக கிடைக்கும் ✔ Home Users-க்கு சிறந்தது ✔ DHCP மூலம் வழங்கப்படும்

🎯 முடிவு

Server, CCTV போன்றவற்றுக்கு Static IP நல்லது. சாதாரண Mobile, Laptop, Home WiFi பயன்பாட்டிற்கு Dynamic IP போதுமானது.
