📡 Ping Connectivity Testing

Basic Network Reachability Verification Tool

📖 What is Ping?

Ping என்பது ஒரு Device மற்றொரு Device-ஐ Network மூலம் Reach செய்ய முடியுமா என்பதை சரிபார்க்க பயன்படும் Diagnostic Command ஆகும்.

Ping = Test Network Connectivity

🎯 Why Use Ping?

✔ Verify Connectivity

✔ Check Network Reachability

✔ Measure Response Time

✔ Detect Network Issues

✔ Initial Troubleshooting Tool

⚙ How Ping Works?

Ping ICMP Echo Request அனுப்புகிறது.

Destination Device ICMP Echo Reply அனுப்புகிறது.

Reply வந்தால் Connectivity உள்ளது என்று அர்த்தம்.

💻 Basic Commands

ping 8.8.8.8

ping google.com

ping 192.168.1.1

📊 Sample Output

Reply from 8.8.8.8:
bytes=32
time=15ms
TTL=118

time = Response Time

TTL = Time To Live
