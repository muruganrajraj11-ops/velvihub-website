Traffic Engineering & MPLS Path Optimization

📖 What is RSVP-TE?

RSVP-TE (Resource Reservation Protocol - Traffic Engineering) என்பது MPLS Network-ல் குறிப்பிட்ட பாதையில் Traffic-ஐ அனுப்ப பயன்படும் Protocol ஆகும்.

RSVP-TE = Explicit Path + Bandwidth Reservation

🎯 Benefits

✔ Traffic Engineering

✔ Bandwidth Reservation

✔ Fast Reroute

✔ Better Network Utilization

🛣 Explicit Path

Traffic எந்த Routers வழியாக செல்ல வேண்டும் என்பதை முன்கூட்டியே நிர்ணயிக்கலாம்.

📦 Resource Reservation

தேவையான Bandwidth-ஐ முன்பதிவு செய்து Traffic Congestion-ஐ குறைக்கலாம்.

⚡ Fast Reroute

Link Failure ஏற்பட்டால் Backup Path-க்கு மிக வேகமாக Traffic மாற்றப்படும்.
