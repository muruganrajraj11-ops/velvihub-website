📝 IP Address MCQ Test

Q1. IP என்பதன் விரிவாக்கம் என்ன?
A) Internet Protocol
B) Internal Program
C) Internet Program
D) Internal Protocol
Answer: A
Q2. IPv4 Address எத்தனை Bit?
A) 16
B) 32
C) 64
D) 128
Answer: B
Q3. IPv6 Address எத்தனை Bit?
A) 32
B) 64
C) 128
D) 256
Answer: C
Q4. Loopback Address எது?
A) 127.0.0.1
B) 192.168.1.1
C) 8.8.8.8
D) 10.0.0.1
Answer: A
Q5. Google DNS எது?
A) 1.1.1.1
B) 4.2.2.2
C) 8.8.8.8
D) 10.0.0.1
Answer: C
