🌐 NAT என்றால் என்ன? – தமிழில் எளிமையாக

NAT (Network Address Translation) என்பது Private IP Address-களை Public IP Address-ஆக மாற்றும் தொழில்நுட்பமாகும்.

இதன் மூலம் பல Devices ஒரு Public IP பயன்படுத்தி Internet Access பெற முடியும்.

🏠 Real-Life Example

ஒரு Apartment-ல் 100 குடும்பங்கள் இருந்தாலும் வெளியே ஒரே Apartment Address இருக்கும்.

அதேபோல் வீட்டில் 10 Devices இருந்தாலும் Internet-க்கு ஒரே Public IP மட்டும் தெரியும்.

⚙️ NAT எப்படி வேலை செய்கிறது?

Laptop, Mobile, Smart TV போன்ற Devices Private IP பயன்படுத்தும்.

Internet-க்கு செல்லும்போது Router அந்த Private IP-ஐ Public IP-ஆக மாற்றும்.

திரும்ப வரும் Response-ஐ சரியான Device-க்கு அனுப்பும்.

📌 Private IP Ranges

  • 10.0.0.0 – 10.255.255.255
  • 172.16.0.0 – 172.31.255.255
  • 192.168.0.0 – 192.168.255.255

📌 NAT வகைகள்

  • Static NAT
  • Dynamic NAT
  • PAT (Port Address Translation)

📊 NAT நன்மைகள்

Benefit Description
IP Conservation Public IP Save செய்கிறது
Security Private Network Hide ஆகும்
Cost Saving குறைந்த Public IP போதும்

🎯 Interview Questions

Q: NAT என்பதன் விரிவாக்கம் என்ன?

A: Network Address Translation

Q: NAT ஏன் பயன்படுத்தப்படுகிறது?

A: Private IP-களை Public IP-ஆக மாற்ற.

Q: Home Router-ல் அதிகமாக எந்த NAT பயன்படுத்தப்படுகிறது?

A: PAT (Port Address Translation)

🚦 Easy Analogy

ஒரு நிறுவனத்தில் உள்ள அனைவரும் ஒரே Reception Number பயன்படுத்துவது போல NAT பல Devices-க்கு ஒரே Public IP-ஐ பயன்படுத்த அனுமதிக்கிறது.

🎯 முடிவு

NAT என்பது Internet பயன்பாட்டில் மிகவும் முக்கியமான தொழில்நுட்பமாகும்.

IPv4 Address பற்றாக்குறையை சமாளிக்க NAT மிகப்பெரிய பங்கு வகிக்கிறது.
