🌐 What is OSI Model?

OSI (Open Systems Interconnection) Model என்பது Network Communication-ஐ 7 Layers ஆக பிரித்து விளக்கும் Model.

OSI = Network Communication Roadmap

7️⃣ OSI Layers

7. Application Layer

6. Presentation Layer

5. Session Layer

4. Transport Layer

3. Network Layer

2. Data Link Layer

1. Physical Layer

📡 Physical Layer

Cable, Fiber, Connector போன்ற Physical Media.

🔗 Data Link Layer

MAC Address மற்றும் Switching Operations.

🌍 Network Layer

IP Address மற்றும் Routing.

🚚 Transport Layer

TCP & UDP Communication.

💻 Upper Layers

Session, Presentation மற்றும் Application Layer User Services-ஐ Handle செய்கின்றன.
