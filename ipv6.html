🚀 Internet Protocol Version 6

Next Generation IP Addressing Technology

📖 What is IPv6?

IPv6 (Internet Protocol Version 6) என்பது IPv4-க்கு அடுத்த தலைமுறை Internet Protocol ஆகும். உலகளவில் IP Address பற்றாக்குறையை தீர்க்க உருவாக்கப்பட்டது.

IPv6 = 128 Bit Addressing System

🎯 Why IPv6?

✔ Unlimited Address Space

✔ Better Security

✔ Improved Routing

✔ Auto Configuration

✔ Future Internet Support

🏗 IPv6 Structure

128 Bits Address

8 Blocks

Hexadecimal Format

Example: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

🌐 Types of IPv6 Addresses

Global Unicast

Link Local

Unique Local

Multicast

Anycast

⚙ How IPv6 Works?

ஒவ்வொரு Device-க்கும் Unique 128-bit Address வழங்கப்படுகிறது.

NAT தேவையில்லாமல் End-to-End Communication சாத்தியமாகிறது.

📊 IPv4 vs IPv6

IPv4 = 32 Bit

IPv6 = 128 Bit

IPv4 = Decimal Format

IPv6 = Hexadecimal Format

IPv4 = Limited Addresses

IPv6 = Huge Address Space

🚀 Advantages

✔ Massive Address Space

✔ Built-in IPSec Support

✔ Better Performance

✔ Auto Configuration

✔ Simplified Header
