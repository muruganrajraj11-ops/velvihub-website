🔀 Variable Length Subnet Mask

Advanced IP Address Allocation Method

📖 What is VLSM?

VLSM (Variable Length Subnet Mask) என்பது ஒரே Network-ல் வெவ்வேறு அளவிலான Subnets உருவாக்க பயன்படும் Subnetting Technique ஆகும்.

VLSM = Different Size Subnets For Better IP Utilization

🎯 Why VLSM?

✔ Efficient IP Address Usage

✔ Flexible Network Design

✔ Reduced IP Wastage

✔ Better Scalability

✔ Professional Address Planning

⚙ How VLSM Works?

ஒவ்வொரு Department அல்லது Network Requirement-க்கு தேவையான Host Count அடிப்படையில் தனித்தனி Subnet Size வழங்கப்படுகிறது.

📊 Example

192.168.1.0/24

HR Department = /26

Finance Department = /27

IT Department = /28

WAN Link = /30
