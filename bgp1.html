📖 What is BGP?

BGP என்பது Internet முழுவதும் பயன்படுத்தப்படும் Exterior Gateway Routing Protocol ஆகும்.

BGP = Border Gateway Protocol

✔ Path Vector Protocol

✔ Internet Routing

✔ Policy Based Routing

✔ Highly Scalable
