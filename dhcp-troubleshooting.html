📖 What is DHCP?

DHCP (Dynamic Host Configuration Protocol) என்பது Devices-க்கு Automatic-ஆக IP Address, Gateway மற்றும் DNS வழங்கும் Protocol ஆகும்.

DHCP = Automatic IP Address Assignment

🎯 Why DHCP?

✔ Automatic IP Assignment

✔ Easy Network Management

✔ Prevent Manual Errors

✔ Centralized Control

✔ Faster Deployment
