📖 What is Google Cloud?

Google Cloud Platform (GCP) என்பது Google வழங்கும் Cloud Computing Platform ஆகும்.

Compute, Storage, Networking, Databases, Artificial Intelligence மற்றும் Big Data Services ஆகியவற்றை Cloud மூலம் வழங்குகிறது.

Google Cloud = Compute + Storage + AI + Analytics
