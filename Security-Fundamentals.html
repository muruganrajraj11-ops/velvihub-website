📖 What is Cyber Security?

Cyber Security என்பது Computers, Servers, Networks, Applications மற்றும் Data ஆகியவற்றை Cyber Attacks-லிருந்து பாதுகாக்கும் தொழில்நுட்பம் ஆகும்.

Cyber Security = Protect + Detect + Respond

🔐 Authentication & Authorization

Authentication = User யார் என்பதை உறுதி செய்தல்.

Authorization = User என்ன செய்யலாம் என்பதை தீர்மானித்தல்.
