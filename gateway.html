🚪 What is Gateway?

Gateway என்பது ஒரு network-ல இருந்து மற்றொரு network-க்கு செல்லும் வழி.

உதாரணம்: உங்க வீட்டு Main Gate வழியாக தான் வெளியே போவீங்க. அதே மாதிரி computer அல்லது mobile internet-க்கு போக Gateway use பண்ணும்.

பொதுவாக Gateway என்பது Router IP address ஆக இருக்கும்.

Example: IP Address : 192.168.1.10 Gateway : 192.168.1.1

Internet-க்கு செல்லும் traffic எல்லாமே முதலில் Gateway-க்கு செல்லும். அதன்பிறகு வெளியே செல்லும்.

Gateway இல்லாமல் Internet access வேலை செய்யாது.
