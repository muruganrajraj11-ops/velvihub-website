🚪 What is Gateway? – தமிழில் எளிமையாக

Gateway என்பது ஒரு Network-லிருந்து மற்றொரு Network-க்கு செல்லும் வழியாக செயல்படும் சாதனம் அல்லது IP Address ஆகும்.

பொதுவாக Home Network-களில் Router-ன் IP Address தான் Gateway ஆக இருக்கும்.

🏠 Real-Life Example

உங்கள் வீட்டின் Main Gate வழியாக தான் வெளியே செல்ல முடியும்.

அதேபோல் Computer அல்லது Mobile Internet-க்கு செல்ல Gateway-ஐ பயன்படுத்துகிறது.

⚙️ Gateway எப்படி வேலை செய்கிறது?

ஒரு Device தனது Local Network-க்கு வெளியே Data அனுப்ப வேண்டுமெனில் முதலில் Gateway-க்கு அனுப்பும்.

Gateway அந்த Data-வை Internet அல்லது மற்றொரு Network-க்கு Forward செய்கிறது.

📌 Example

IP Address : 192.168.1.10

Subnet Mask : 255.255.255.0

Gateway : 192.168.1.1

இந்த நிலையில் Device Internet Access பெற 192.168.1.1 Gateway-ஐ பயன்படுத்தும்.

🚀 Gateway நன்மைகள்

  • ✅ Internet Access வழங்கும்
  • ✅ Different Networks-ஐ இணைக்கும்
  • ✅ Traffic Routing செய்கிறது
  • ✅ Network Communication-ஐ எளிதாக்குகிறது

🎯 Interview Questions

Q: Gateway என்றால் என்ன?

A: ஒரு Network-லிருந்து மற்றொரு Network-க்கு செல்லும் பாதை.

Q: Home Network-ல் Gateway பொதுவாக என்ன?

A: Router IP Address.

Q: Gateway இல்லையெனில் என்ன ஆகும்?

A: Internet Access வேலை செய்யாது.

🚦 Easy Analogy

வீட்டின் Main Gate வழியாக தான் வெளியே செல்ல முடியும்.

அதேபோல் Gateway வழியாக தான் Device Internet-க்கு செல்ல முடியும்.

🎯 முடிவு

Gateway என்பது Network Communication மற்றும் Internet Access-க்கு மிகவும் முக்கியமான அடிப்படை அமைப்பாகும்.
