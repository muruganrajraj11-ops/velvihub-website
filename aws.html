📖 What is AWS?

AWS (Amazon Web Services) என்பது Amazon வழங்கும் Cloud Computing Platform ஆகும்.

Servers, Storage, Networking, Databases மற்றும் பல Cloud Services-ஐ இணையத்தின் மூலம் வழங்குகிறது.

AWS = Compute + Storage + Network + Security
