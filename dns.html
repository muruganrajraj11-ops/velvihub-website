🌐 What is DNS? – தமிழில் எளிமையாக

DNS (Domain Name System) என்பது இணையத்தில் Website பெயர்களை IP Address-ஆக மாற்றும் Service ஆகும்.

நாம் Browser-ல் google.com என்று Type செய்கிறோம். ஆனால் Computer-க்கு பெயர் புரியாது. அதற்கு IP Address தான் புரியும்.

DNS அந்த Website பெயரை சரியான IP Address-ஆக மாற்றி Connect செய்கிறது.

🏠 Real-Life Example

DNS = Phone Contact List

Website Name = Person Name

IP Address = Phone Number

நீங்கள் Ravi என்று Contact-ஐ Click செய்தால் Call செல்லும். அதேபோல் DNS Website Name-ஐ IP Address-ஆக மாற்றி சரியான Server-க்கு Connect செய்கிறது.

📌 Popular DNS Servers

  • Google DNS - 8.8.8.8
  • Google DNS - 8.8.4.4
  • Cloudflare DNS - 1.1.1.1
  • Cloudflare DNS - 1.0.0.1

⚙️ DNS எப்படி வேலை செய்கிறது?

1. User Browser-ல் Website Name Type செய்கிறார்.

2. DNS அந்த பெயருக்கான IP Address-ஐ தேடுகிறது.

3. சரியான IP கிடைத்ததும் Website Open ஆகிறது.

🎯 Interview Questions

Q: DNS என்பதன் விரிவாக்கம் என்ன?

A: Domain Name System

Q: DNS-ன் முக்கிய பணி என்ன?

A: Domain Name-ஐ IP Address-ஆக மாற்றுவது.

Q: Google DNS Address என்ன?

A: 8.8.8.8 மற்றும் 8.8.4.4

🎯 முடிவு

DNS இல்லையென்றால் Website Name மூலம் Internet பயன்படுத்த முடியாது.

Internet-ன் மிகவும் முக்கியமான அடிப்படை Service-களில் DNS ஒன்றாகும்.
