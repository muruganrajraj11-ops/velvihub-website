🌐 What is DNS?

DNS என்றால் Domain Name System.

இது இணையத்தில் website பெயர்களை IP Address-ஆக மாற்றும் ஒரு service.

உதாரணம்: நாம் google.com என்று type செய்கிறோம். ஆனால் computer-க்கு பெயர் புரியாது. அதற்கு IP Address தான் புரியும்.

DNS அந்த website பெயரை சரியான IP Address-ஆக மாற்றி connect செய்கிறது.

Real Life Example: DNS = Phone Contact List Website Name = Person Name IP Address = Phone Number

Popular DNS: 8.8.8.8 8.8.4.4 (Google DNS)
