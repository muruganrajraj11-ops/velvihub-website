🔀 Network Switch என்றால் என்ன? – தமிழில் எளிமையாக

Network Switch என்பது ஒரு LAN (Local Area Network) உள்ளே பல Devices-ஐ இணைக்கும் Networking Device ஆகும்.

Router Internet-க்கு வழிகாட்டினால், Switch ஒரே Network-க்குள் உள்ள Devices ஒன்றுக்கொன்று தொடர்பு கொள்ள உதவுகிறது.

🏢 Real-Life Example

ஒரு Office-ல் 20 Computer-கள் இருக்கின்றன என்று வைத்துக்கொள்ளுங்கள்.

அனைத்து Computer-களும் ஒரு Switch-க்கு Connect செய்யப்பட்டிருக்கும்.

Computer A → Computer B-க்கு Data அனுப்பினால், Switch அந்த Data-வை நேராக Computer B-க்கு மட்டும் அனுப்பும்.

மற்ற Devices-க்கு அனுப்பாது.

⚙️ Switch எப்படி வேலை செய்கிறது?

Switch ஒவ்வொரு Device-ன் MAC Address-ஐ கற்றுக்கொள்கிறது.

அதை MAC Address Table-ல் சேமித்து வைக்கிறது.

பிறகு Data வந்தவுடன் எந்த Port-க்கு அனுப்ப வேண்டும் என்று முடிவு செய்கிறது.

📋 MAC Address Table என்றால் என்ன?

Switch-ன் Memory-யில் இருக்கும் ஒரு பட்டியல்.

எந்த Device எந்த Port-ல் உள்ளது என்பதை இதில் சேமித்து வைத்திருக்கும்.

இதனால் Data சரியான இடத்துக்கு விரைவாக செல்கிறது.

📊 Switch vs Hub

Feature Hub Switch
Data Forwarding All Devices Specific Device
Speed Slow Fast
Security Low High
Efficiency Low High

⚡ Switch-ன் நன்மைகள்

  • ✅ Network Speed அதிகரிக்கும்
  • ✅ Unnecessary Traffic குறையும்
  • ✅ Multiple Devices Connect செய்யலாம்
  • ✅ Hub-ஐ விட அதிக Security
  • ✅ Efficient Communication

🎯 Interview Questions

Q: Switch எந்த Address-ஐ பயன்படுத்தி Data Forward செய்கிறது?

A: MAC Address

Q: Switch எந்த OSI Layer-ல் வேலை செய்கிறது?

A: Layer 2 (Data Link Layer)

Q: Switch மற்றும் Hub இடையேயான முக்கிய வித்தியாசம் என்ன?

A: Switch தேவையான Device-க்கு மட்டும் Data அனுப்பும்.

🚦 Easy Analogy

ஒரு Traffic Police எந்த வாகனம் எந்த வழியில் செல்ல வேண்டும் என்று வழிகாட்டுவார்.

அதேபோல் Switch எந்த Data எந்த Device-க்கு செல்ல வேண்டும் என்பதை முடிவு செய்கிறது.

🎯 முடிவு

Network Switch என்பது LAN Network-களில் மிகவும் முக்கியமான Networking Device ஆகும்.

இது Devices-களுக்கிடையே வேகமான மற்றும் திறமையான Communication-ஐ வழங்குகிறது.
