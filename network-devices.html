📖 What are Network Devices?

Network Devices என்பது Network-ல் Data Communication செய்ய பயன்படும் Hardware சாதனங்கள்.

Network Devices = Network Traffic Controllers

🌐 Router

வேறு வேறு Networks-ஐ இணைத்து Routing செய்கிறது.

🔀 Switch

ஒரே LAN-ல் உள்ள Devices-ஐ MAC Address மூலம் இணைக்கிறது.

🔥 Firewall

Network Security மற்றும் Traffic Filtering செய்கிறது.

📶 Access Point

Wireless Wi-Fi Connectivity வழங்குகிறது.

🖥 Server

Network Services மற்றும் Applications வழங்குகிறது.

💻 Client Device

Computer, Laptop, Mobile போன்ற End User Devices.
