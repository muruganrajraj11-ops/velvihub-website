📖 What is IDS?

IDS (Intrusion Detection System) என்பது Network அல்லது System-ல் சந்தேகத்திற்கிடமான செயல்பாடுகளை கண்டறியும் Security Tool ஆகும்.

IDS = Detect & Alert

📖 What is IPS?

IPS (Intrusion Prevention System) என்பது Attack-களை கண்டறிவதுடன் மட்டுமல்லாமல் அவற்றை தடுக்கவும் செய்கிறது.

IPS = Detect + Block
