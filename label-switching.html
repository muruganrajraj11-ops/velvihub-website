📖 What is Label Switching?

MPLS Network-ல் Packet Routing Table மூலம் அல்ல, Label மூலம் Forward செய்யப்படுகிறது.

Label = Fast Path Identifier

🔹 LER

Label Edge Router Packet-க்கு Label சேர்க்கவும் நீக்கவும் செய்கிறது.

🔹 LSR

Label Switch Router Label அடிப்படையில் Packet-ஐ Forward செய்கிறது.

🛣 LSP

Label Switched Path என்பது MPLS Network-ல் Packet செல்லும் பாதை.
