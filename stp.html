🌳 STP (Spanning Tree Protocol) – தமிழில் எளிமையாக

STP (Spanning Tree Protocol) என்பது Switch Network-களில் Loop ஏற்படாமல் தடுக்க பயன்படும் Protocol ஆகும்.

Layer 2 Network-ல் ஏற்படும் Broadcast Storm மற்றும் Switching Loop-களை தடுக்க STP பயன்படுத்தப்படுகிறது.

🏠 Real-Life Example

ஒரு நகரத்தில் இரண்டு சாலைகள் ஒரே இடத்துக்கு செல்லும் போது Traffic Loop ஏற்பட்டால் குழப்பம் வரும்.

அதேபோல் Network-ல் Loop ஏற்பட்டால் Broadcast Storm உருவாகும்.

அதைத் தடுக்க STP பயன்படுகிறது.

⚠️ Network Loop என்றால் என்ன?

இரண்டு அல்லது அதற்கு மேற்பட்ட Switch-கள் பல Link-களால் இணைக்கப்பட்டு Packet மீண்டும் மீண்டும் சுற்றுவது Network Loop ஆகும்.

💥 Broadcast Storm

Loop காரணமாக Broadcast Packets தொடர்ந்து Network-ல் சுற்றிக்கொண்டே இருக்கும்.

இதனால் Network Slow ஆகலாம் அல்லது முழுமையாக Down ஆகலாம்.

⚙️ STP எப்படி வேலை செய்கிறது?

STP முதலில் ஒரு Root Bridge-ஐ தேர்வு செய்கிறது.

பிறகு தேவையில்லாத Link-களை Block செய்து Loop இல்லாத Path உருவாக்குகிறது.

📌 STP முக்கிய Terms

  • Root Bridge
  • Root Port
  • Designated Port
  • Blocked Port

📊 STP Port States

State Purpose
Blocking Loop Prevention
Listening STP Calculation
Learning MAC Address Learning
Forwarding Normal Traffic Forwarding

🎯 Interview Questions

Q: STP என்பதன் விரிவாக்கம் என்ன?

A: Spanning Tree Protocol

Q: STP ஏன் பயன்படுத்தப்படுகிறது?

A: Network Loop Prevention.

Q: STP எந்த Layer-ல் வேலை செய்கிறது?

A: Layer 2 (Data Link Layer).

🚦 Easy Analogy

ஒரு Traffic Police தேவையில்லாத சாலையை மூடி Traffic Jam-ஐ தடுக்கிறார்.

அதேபோல் STP தேவையில்லாத Network Path-ஐ Block செய்கிறது.

🎯 முடிவு

STP என்பது Switch Network-களில் Loop Prevention மற்றும் Network Stability-க்கு மிகவும் முக்கியமான Protocol ஆகும்.

RSTP போன்ற மேம்பட்ட Protocol-களின் அடிப்படை தொழில்நுட்பமாகவும் STP செயல்படுகிறது.
