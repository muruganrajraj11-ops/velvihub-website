📖 What is Virtualization?

Virtualization என்பது ஒரு Physical Server-ஐ பல Virtual Machines (VMs) ஆக பிரித்து இயக்கும் தொழில்நுட்பமாகும்.

One Physical Server → Multiple Virtual Servers

✔ Better Resource Utilization

✔ Cost Reduction

✔ Easy Management

✔ Faster Deployment

✔ High Availability
