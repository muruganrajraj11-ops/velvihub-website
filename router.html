🌐 What is Router? – தமிழில் எளிமையாக

Router என்பது இரண்டு அல்லது அதற்கு மேற்பட்ட Networks-ஐ இணைக்கும் Networking Device ஆகும்.

இது ஒரு Network-லிருந்து மற்றொரு Network-க்கு Data Packets-ஐ சரியான பாதையில் அனுப்புகிறது.

🏠 Real-Life Example

உங்கள் வீட்டில் Mobile, Laptop, Smart TV ஆகியவை WiFi மூலம் Router-க்கு Connect ஆகும்.

Router தான் Internet-லிருந்து வரும் Data-வை சரியான Device-க்கு அனுப்புகிறது.

📶 Router என்ன செய்கிறது?

  • ✅ Internet Connection வழங்குகிறது
  • ✅ DHCP மூலம் IP Address கொடுக்கிறது
  • ✅ DNS Requests அனுப்புகிறது
  • ✅ Network Traffic-ஐ நிர்வகிக்கிறது
  • ✅ Different Networks-ஐ இணைக்கிறது

🎯 Interview Questions

Q: Router-ன் முக்கிய பணி என்ன?

A: Different Networks-ஐ இணைத்து Data Routing செய்வது.

Q: Router எந்த OSI Layer-ல் வேலை செய்கிறது?

A: Layer 3 (Network Layer).

Q: Router எதை அடிப்படையாக கொண்டு Forwarding செய்கிறது?

A: IP Address.

🚦 Easy Analogy

ஒரு Traffic Police எந்த வாகனம் எந்த வழியில் செல்ல வேண்டும் என்று வழிகாட்டுவார்.

அதேபோல் Router எந்த Data எந்த Network-க்கு செல்ல வேண்டும் என்பதை முடிவு செய்கிறது.

🎯 முடிவு

Router என்பது Local Network மற்றும் Internet இடையே இணைப்பை வழங்கும் மிக முக்கியமான Networking Device ஆகும்.

வீட்டு WiFi முதல் பெரிய ISP Networks வரை Router முக்கிய பங்கு வகிக்கிறது.
