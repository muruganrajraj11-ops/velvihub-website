DNS Issues Detection & Resolution

📖 What is DNS?

DNS (Domain Name System) என்பது Domain Name-ஐ IP Address-ஆக மாற்றும் Internet Service ஆகும்.

DNS = Internet Phone Book

🎯 Why DNS is Important?

✔ Website Access

✔ Email Services

✔ Internet Browsing

✔ Application Connectivity

✔ Domain Resolution

⚙ How DNS Works?

User google.com என Type செய்தால் DNS Server அதற்கான IP Address-ஐ கண்டுபிடித்து வழங்குகிறது.

பிறகு Browser அந்த IP Address-ஐ பயன்படுத்தி Website-ஐ திறக்கிறது.

💻 DNS Commands

nslookup google.com

nslookup openai.com

ipconfig /flushdns

ipconfig /displaydns
