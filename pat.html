🔄 PAT என்றால் என்ன? – தமிழில் எளிமையாக

PAT (Port Address Translation) என்பது NAT-ன் மிகவும் பரவலாக பயன்படுத்தப்படும் ஒரு வகை ஆகும்.

பல Private IP Devices ஒரு Public IP Address-ஐ Port Numbers மூலம் பகிர்ந்து பயன்படுத்த PAT உதவுகிறது.

🏠 Real-Life Example

ஒரு Apartment-க்கு ஒரே முகவரி இருக்கும்.

ஆனால் ஒவ்வொரு வீட்டுக்கும் தனித்தனி Flat Number இருக்கும்.

அதேபோல் PAT-ல் Public IP ஒன்று இருந்தாலும் Port Number மூலம் Device-கள் தனித்தனியாக அடையாளம் காணப்படும்.

⚙️ PAT எப்படி வேலை செய்கிறது?

192.168.1.10 → 203.0.113.1:5001

192.168.1.20 → 203.0.113.1:5002

192.168.1.30 → 203.0.113.1:5003

இவ்வாறு ஒரே Public IP-க்கு வெவ்வேறு Port Numbers வழங்கப்படும்.

📌 PAT ஏன் முக்கியம்?

  • ✅ Public IP Save செய்கிறது
  • ✅ Home Broadband-ல் அதிகம் பயன்படுத்தப்படுகிறது
  • ✅ IPv4 Address Shortage-ஐ சமாளிக்க உதவுகிறது
  • ✅ Multiple Devices Support

📊 NAT vs PAT

Feature NAT PAT
Public IP Usage One or Many Single Public IP
Port Translation No Yes
Efficiency Medium High

🎯 Interview Questions

Q: PAT என்பதன் விரிவாக்கம் என்ன?

A: Port Address Translation

Q: PAT-ன் முக்கிய நன்மை என்ன?

A: ஒரு Public IP மூலம் பல Devices Internet Access பெற முடியும்.

Q: Home Router-ல் அதிகம் பயன்படுத்தப்படுவது NAT ஆ PAT ஆ?

A: PAT

🚦 Easy Analogy

ஒரு பெரிய அலுவலகத்திற்கு ஒரே Telephone Number இருக்கும்.

ஆனால் ஒவ்வொரு ஊழியருக்கும் Extension Number இருக்கும்.

அதேபோல் PAT Public IP + Port Number பயன்படுத்துகிறது.

🎯 முடிவு

PAT என்பது NAT-ன் மிகவும் பிரபலமான வடிவமாகும்.

Home Broadband, Enterprise Networks மற்றும் ISP Networks-ல் இது பரவலாக பயன்படுத்தப்படுகிறது.
