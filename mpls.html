☁️ MPLS என்றால் என்ன? – தமிழில் எளிமையாக

MPLS (Multi Protocol Label Switching) என்பது வேகமான Data Forwarding செய்ய பயன்படும் Networking Technology ஆகும்.

சாதாரண IP Routing-ல் Router IP Address-ஐ பார்க்கும். MPLS-ல் Router Label-ஐ பார்த்து Forward செய்கிறது.

🏠 Real-Life Example

ஒரு Courier Parcel-க்கு Destination Sticker ஒட்டப்பட்டிருக்கும்.

ஒவ்வொரு Office-லும் முழு Address படிக்காமல் Sticker பார்த்து அனுப்பிவிடும்.

அதேபோல் MPLS Router-கள் Label பார்த்து Data அனுப்புகின்றன.

⚙️ MPLS எப்படி வேலை செய்கிறது?

1. Packet Network-க்கு நுழையும் போது Label சேர்க்கப்படும்.

2. நடுவில் உள்ள Routers Label-ஐ மட்டும் பார்க்கும்.

3. இறுதியில் Label நீக்கப்பட்டு Destination-க்கு அனுப்பப்படும்.

📌 MPLS Components

  • LER – Label Edge Router
  • LSR – Label Switch Router
  • LSP – Label Switched Path

🚀 MPLS நன்மைகள்

  • ✅ Faster Forwarding
  • ✅ Traffic Engineering
  • ✅ QoS Support
  • ✅ VPN Support
  • ✅ Better WAN Performance

📊 MPLS vs Traditional Routing

Feature MPLS Traditional IP Routing
Forwarding Label Based IP Based
Speed Fast Normal
Traffic Engineering Supported Limited

🎯 Interview Questions

Q: MPLS என்பதன் விரிவாக்கம் என்ன?

A: Multi Protocol Label Switching

Q: MPLS-ல் Packet எப்படி Forward செய்யப்படுகிறது?

A: Label அடிப்படையில்.

Q: MPLS-ன் முக்கிய நன்மை என்ன?

A: Fast Forwarding மற்றும் Traffic Engineering.

🚦 Easy Analogy

ஒரு Airport-ல் Baggage Tag வைத்து Bags அனுப்புவது போல MPLS Label வைத்து Packets அனுப்புகிறது.

🎯 முடிவு

MPLS என்பது Telecom மற்றும் ISP Networks-ல் மிகவும் முக்கியமான Technology ஆகும்.

RailTel, BSNL, Airtel போன்ற நிறுவனங்களில் MPLS பரவலாக பயன்படுத்தப்படுகிறது.
