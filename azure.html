📖 What is Microsoft Azure?

Microsoft Azure என்பது Microsoft வழங்கும் Cloud Computing Platform ஆகும்.

Virtual Machines, Networking, Storage, Databases மற்றும் AI Services ஆகியவற்றை Cloud-ல் வழங்குகிறது.
