🌐 IPv4 vs IPv6 – தமிழில் எளிமையாக

Internet-ல் உள்ள ஒவ்வொரு Device-க்கும் ஒரு IP Address தேவை.

இன்று உலகில் அதிகம் பயன்படுத்தப்படுவது IPv4. அதன் புதிய பதிப்பு IPv6 ஆகும்.

🌐 IPv4 என்றால் என்ன?

IPv4 (Internet Protocol Version 4) என்பது தற்போதைய Internet-ல் அதிகமாக பயன்படுத்தப்படும் Addressing System.

உதாரணம்: 192.168.1.10

இதில் 32-bit Address பயன்படுத்தப்படுகிறது.

🚀 IPv6 என்றால் என்ன?

IPv6 (Internet Protocol Version 6) என்பது IPv4 Address பற்றாக்குறையை தீர்க்க உருவாக்கப்பட்டது.

உதாரணம்: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

இதில் 128-bit Address பயன்படுத்தப்படுகிறது.

🔢 IPv4 Address Format

IPv4 = 4 Octets

192.168.1.10

ஒவ்வொரு Octet-லும் 0 முதல் 255 வரை மதிப்புகள் இருக்கும்.

🔢 IPv6 Address Format

IPv6 = 8 Groups

2001:db8:85a3::8a2e:370:7334

Hexadecimal வடிவில் எழுதப்படுகிறது.

📊 IPv4 vs IPv6 Comparison

Feature IPv4 IPv6
Address Length 32 Bit 128 Bit
Format Decimal Hexadecimal
Example 192.168.1.1 2001:db8::1
Addresses Available Limited Very Large
NAT Required Yes No

🌍 IPv6 ஏன் உருவாக்கப்பட்டது?

Internet-ல் Devices எண்ணிக்கை அதிகரித்ததால் IPv4 Address-கள் குறையத் தொடங்கின.

இந்த பிரச்சினையை தீர்க்க IPv6 உருவாக்கப்பட்டது.

🏠 Real-Life Example

IPv4 என்பது ஒரு சிறிய நகரத்தில் இருக்கும் வீட்டு எண்கள் போன்றது.

IPv6 என்பது உலகிலுள்ள அனைத்து வீடுகளுக்கும் தனித்தனி எண்கள் கொடுக்கக்கூடிய மிகப்பெரிய Address System போன்றது.

🎯 Interview Questions

Q: IPv4 எத்தனை Bit Address பயன்படுத்துகிறது?

A: 32 Bit


Q: IPv6 எத்தனை Bit Address பயன்படுத்துகிறது?

A: 128 Bit


Q: IPv6 ஏன் உருவாக்கப்பட்டது?

A: IPv4 Address Shortage-ஐ தீர்க்க.

🚦 Easy Analogy

IPv4 = சிறிய Apartment Address System

IPv6 = முழு உலகத்திற்கும் போதுமான Address System

🎯 முடிவு

IPv4 தற்போது பரவலாக பயன்படுத்தப்படுகிறது.

ஆனால் எதிர்கால Internet வளர்ச்சிக்காக IPv6 மிகவும் முக்கியமான தொழில்நுட்பமாகும்.
