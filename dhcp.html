📡 What is DHCP? – தமிழில் எளிமையாக

DHCP (Dynamic Host Configuration Protocol) என்பது Network-ல் Device-களுக்கு தானாக IP Address வழங்கும் Service ஆகும்.

நாம் WiFi-க்கு Mobile அல்லது Laptop Connect பண்ணும் போது IP Address Manual-ஆக கொடுக்க வேண்டியதில்லை.

Router-ல் இருக்கும் DHCP Service தானாக:

  • IP Address
  • Subnet Mask
  • Default Gateway
  • DNS Server

போன்ற தகவல்களை வழங்கும்.

🏨 Real-Life Example

ஒரு Hotel-க்கு வரும் ஒவ்வொரு விருந்தினருக்கும் Reception Room Number கொடுக்கும்.

அதேபோல் DHCP Network-ல் Connect ஆகும் ஒவ்வொரு Device-க்கும் IP Address வழங்குகிறது.

📶 Router Example

Router IP : 192.168.1.1

DHCP Range : 192.168.1.100 முதல் 192.168.1.200 வரை

இந்த Range-லிருந்து Router தானாக IP Assign செய்யும்.

🚀 DHCP நன்மைகள்

  • ✅ Automatic IP Assignment
  • ✅ Easy Network Management
  • ✅ Reduced Configuration Errors
  • ✅ Centralized Control
  • ✅ Time Saving

🎯 Interview Questions

Q: DHCP என்பதன் விரிவாக்கம் என்ன?

A: Dynamic Host Configuration Protocol

Q: DHCP என்ன வழங்குகிறது?

A: IP Address, Gateway, DNS மற்றும் Subnet Mask

Q: DHCP Server இல்லையென்றால் என்ன ஆகும்?

A: Manual IP Configuration செய்ய வேண்டும்.

🎯 முடிவு

DHCP என்பது Network Administrator-ன் வேலையை எளிதாக்கும் Automatic IP Address Assignment Service ஆகும்.
