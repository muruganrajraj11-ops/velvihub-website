📖 What is Traceroute?

Traceroute என்பது Source Device-லிருந்து Destination Device வரை Packet எந்த Routers வழியாக செல்கிறது என்பதை கண்டறியும் Diagnostic Tool ஆகும்.

Traceroute = Discover Every Hop Between Source and Destination

🎯 Why Use Traceroute?

✔ Identify Routing Path

✔ Detect Network Delays

✔ Locate Faulty Routers

✔ Analyze Internet Routes

✔ Advanced Troubleshooting

⚙ How Traceroute Works?

Traceroute TTL (Time To Live) Value-ஐ படிப்படியாக உயர்த்தி Packets அனுப்புகிறது.

ஒவ்வொரு Router-மும் TTL Expire ஆனபோது பதில் அனுப்பும்.

இதனால் முழு Route தெரியும்.

💻 Basic Commands

Windows:

tracert google.com

tracert 8.8.8.8

Linux:

traceroute google.com

📊 Sample Output

1   1ms    Router
2   8ms    ISP Router
3   15ms   Core Router
4   20ms   Google Network
