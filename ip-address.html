🌐 What is IP Address?

IP address என்பது internet-ல ஒரு device-க்கு இருக்கும் தனி address.

நம்ம வீட்டு address மாதிரி தான்.

Mobile, laptop, router எல்லாத்துக்கும் IP இருக்கும்.

அதனால devices ஒன்றுக்கொன்று connect ஆகும்.
