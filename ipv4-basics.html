📖 What is IPv4?

IPv4 (Internet Protocol Version 4) என்பது Network-ல் உள்ள Devices ஒன்றுக்கொன்று தொடர்பு கொள்ள பயன்படுத்தப்படும் Addressing Protocol ஆகும்.

IPv4 = Unique Address For Every Device

🎯 Why IPv4?

✔ Device Identification

✔ Network Communication

✔ Internet Connectivity

✔ Routing Support

🏗 IPv4 Structure

IPv4 = 32 Bits

4 Octets

Example: 192.168.1.1

📊 IPv4 Classes

Class A : 1 - 126

Class B : 128 - 191

Class C : 192 - 223

Class D : Multicast

Class E : Research

🌍 Private IP Addresses

172.16.0.0 - 172.31.255.255

192.168.0.0/16
